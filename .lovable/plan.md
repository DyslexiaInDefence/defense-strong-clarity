

# Dyslexia in Defence — Website Plan

## Design Foundation
- **Primary palette:** Light blue base with a flowing accent ribbon incorporating yellow, blue, red, and green
- **Style:** Clean, modern, rounded corners, soft shadows, subtle gradients — professional and defence-adjacent without being militaristic
- **Typography:** Large, dyslexia-friendly font (OpenDyslexic or similar readable sans-serif), strong heading hierarchy
- **Tone:** Professional, empowering, strength-based, clear and direct

## Accessibility Controls (in Navigation Bar)
Three toggle controls placed in the top navigation for easy access:
- 🌙 **Dark mode toggle**
- 🔤 **Text size increase toggle**
- 🎯 **Reduce motion toggle**

All controls persistent across pages with preferences saved locally.

## Page Structure (Single-Page Scrolling Site)

### 1. Sticky Navigation Bar
- Logo on left ("Dyslexia in Defence")
- Menu links: Home, About, Resources, Support, Join, Contact
- Accessibility toggles (dark mode, text size, reduce motion)
- Prominent "Get Involved" CTA button
- Sticky on scroll with subtle backdrop blur

### 2. Hero Section
- Bold headline: **"Dyslexia in Defence"**
- Clear one-sentence mission subheading
- Two CTA buttons: "Join the Network" (primary) + "Learn More" (secondary)
- Flowing multi-colour accent ribbon (yellow, blue, red, green) as decorative element
- Clean, accessible imagery/illustration area

### 3. About Section
- Three clear subsections using bullet points:
  - What dyslexia is
  - Why it matters in defence
  - Strengths associated with neurodiversity
- Short paragraphs, generous spacing, strong headings

### 4. Resources Section
- Card-based grid layout (2x2 on desktop, stacked on mobile)
- Four resource cards with icons:
  - 📋 Guidance
  - 🔬 Research
  - 🏢 Workplace Adjustments
  - 👥 Leadership Support
- Each card has a clear "View Resource" button

### 5. Support & Community Section
- Emphasis on psychological safety, confidential support, peer connection
- Benefits highlighted with bold text and bullet points
- Warm, inviting visual treatment

### 6. Get Involved Section
- Large call-to-action banner
- Four clear next steps presented as cards or columns:
  - Join · Partner · Sponsor · Volunteer
- Each with brief description and action button

### 7. Footer
- Contact details
- Social media links (placeholder icons)
- Privacy statement link
- Accessibility statement link
- Commitment to inclusion statement
- Fully accessible layout

## Technical Approach
- Fully responsive (mobile-first)
- WCAG AA compliant (contrast ratios, ARIA labels, semantic HTML, landmarks)
- Alt text placeholders on all images
- Minimal animation (respects reduce motion preference)
- All frontend — no backend required initially
- Performance optimised with clean code
